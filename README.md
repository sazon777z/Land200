# LC200 Smart Desktop Watch

Настольные смарт-часы в корпусе масштабной модели автомобиля Land Cruiser 200 с функцией отображения времени, погоды и будильника с аудиоэффектами двигателей.

## 🚗 Обзор проекта

Умные часы в виде модели Land Cruiser 200, которые синхронизируют время через интернет, показывают погоду и срабатывают как будильник с реалистичными звуками двигателей автомобилей.

## 📋 Технические характеристики

### Hardware
- **Контроллер:** ESP32-C3 Super Mini
- **Дисплей:** 1.3" IPS LCD 240x240 (ST7789)
- **Аудио:** DFPlayer Mini MP3
- **Адресная подсветка:** 4x WS2812B (эффект радуги)
- **Освещение:** Стандартные светодиоды (передние фары, задние фонари)
- **Управление:** 1 тактовая кнопка

### Pinout Mapping

| Компонент       | Пин ESP32-C3 | Тип сигнала | Описание                  |
|-----------------|--------------|-------------|---------------------------|
| ST7789 SCK      | GPIO 6       | SPI CLK     |                           |
| ST7789 MOSI     | GPIO 7       | SPI MOSI    |                           |
| ST7789 CS       | GPIO 4       | Output      |                           |
| ST7789 DC       | GPIO 5       | Output      |                           |
| ST7789 RST      | GPIO 8       | Output      |                           |
| ST7789 BL       | GPIO 10      | PWM/Output  | Подсветка экрана          |
| WS2812B DIN     | GPIO 2       | Digital Out | Цепочка из 4 диодов       |
| DFPlayer RX     | GPIO 21 (TX) | UART        | Через резистор 1кОм       |
| DFPlayer TX     | GPIO 20 (RX) | UART        |                           |
| Front LEDs      | GPIO 0       | Output/PWM  | Передние фары             |
| Rear LEDs       | GPIO 1       | Output/PWM  | Задние фонари             |
| Button          | GPIO 3       | Input Pullup| Прерывание по нажатию     |

## 🔧 Используемые библиотеки

- **Arduino_GFX** - Драйвер дисплея
- **WiFi.h** - Подключение к сети WiFi
- **HTTPClient.h** - HTTP запросы для погоды
- **NTPClient.h** - Синхронизация времени
- **FastLED / Adafruit_NeoPixel** - Управление WS2812B
- **DFRobotDFPlayerMini** - Управление MP3 плеером
- **ESPAsyncWebServer** - Веб-сервер для настройки
- **LittleFS** - Файловая система для хранения UI

## 🚀 Быстрый старт

### Требования
- Arduino IDE 2.x
- ESP32 Board Package
- Все библиотеки из списка выше

### Установка прошивки
1. Клонируйте репозиторий
2. Откройте `Land_Cruiser.ino` в Arduino IDE
3. Выберите плату: ESP32C3 Dev Module
4. Настройте параметры WiFi и API ключей в `config.h`
5. Загрузите прошивку
6. Загрузите файловую систему через `upload_fs.bat` или инструмент Arduino IDE

### Первая настройка
1. При первом включении создастся точка доступа WiFi
2. Подключитесь к сети и откройте веб-интерфейс
3. Настройте параметры сети и временной пояс
4. Установите время будильника и выберите звук

## 🎯 Функции

### Дисплей
- Цифровые/аналоговые часы
- Температура и иконки погоды
- Статус системы

### Веб-интерфейс
- **Dashboard:** Статус системы
- **Настройки времени:** Часовой пояс, регион для погоды
- **Управление будильником:** Время, выбор звука, тестовое воспроизведение
- **Система:** Обновление прошивки, перезагрузка

### Аудиоэффекты
- Звуки двигателей V8, V6, Diesel
- Воспроизведение при срабатывании будильника

### Визуальные эффекты
- **Режим ожидания:** Плавная радуга на WS2812B
- **Режим будильника:** Резкие вспышки красного/белого
- **Мигание фарами:** В противофазе 2 Гц

## ⚡ Питание

Используйте стабильный источник питания 5V 2A для обеспечения работы всех компонентов, особенно DFPlayer и WS2812B.

## 🔄 Алгоритм работы

1. **Старт:** Инициализация периферии → Подключение к WiFi (если нет — режим AP)
2. **Цикл ожидания:** 
   - Обновление времени каждую секунду
   - Обновление погоды каждые 15-30 минут
   - WS2812B вращают цветовой спектр
3. **Будильник:**
   - Воспроизведение MP3 трека
   - Мигание фарами
   - Ожидание нажатия кнопки или деактивации в веб-интерфейсе
4. **Кнопка:** Короткое нажатие — стоп будильник
